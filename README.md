## Project Overview

**Paradise Nursery** is a fictional plant store website designed to provide users with:

- Easy browsing of plant categories.
- Interactive shopping cart functionality.
- Smooth scroll and sticky navigation.
- A dynamic landing page with a looping video background.
- Mobile-first responsive design.

The project demonstrates front-end **React** development, state management with **Redux**, and modern **CSS** techniques.

---

## Features

- **Landing Page** with background video and welcome message.
- **Product Listing** with categories:
  - Air Purifying Plants
  - Aromatic Fragrant Plants
  - Insect Repellent Plants
  - Medicinal Plants
  - Low Maintenance Plants
- **Shopping Cart** with add/remove functionality.
- **Sticky Navbar** with smooth scrolling and a “Back to Top” button.
- **Responsive Design** for mobile and desktop screens.
- **Optimized Background Video** for faster page loading.

---

## Technologies

- **Frontend:** React, JavaScript, HTML, CSS
- **State Management:** Redux
- **Icons:** React Icons
- **Video Background:** MP4 with `autoPlay`, `loop`, `muted` attributes
- **Styling:** CSS Flexbox, Grid, and responsive design techniques
